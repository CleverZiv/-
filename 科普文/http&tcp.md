> 自定义 http header 设计的主要目的是为了提供附加信息，如在企业开发中使用到的泳道技术，会需要在发送请求的header中附加泳道信息，或者是为了方便排查日志，在一个请求中带上 logId，这样整个业务流程都可以带上同一个 logId，对于排查整个流程的日志非常有帮助
